Fetch API - 
Biblioteca nativa do browser.
Oferece alternativa ao XMLHttpRequest() e JQuery.ajax()
- Suporte a Service Workers

Diferenças:
	Não envia nem recebe cookies(precisa definir a opção credentials
	Não rejeita o status de erro do HTTP.



AXIOS
-LIB de HTTP API
- Cross-browser
- pode monitorar o prograsso de um request
- Melhor tratamento de erros
- Melhor teste

Instalar a biblioteca
yarn add axios

import axios from 'axios';
