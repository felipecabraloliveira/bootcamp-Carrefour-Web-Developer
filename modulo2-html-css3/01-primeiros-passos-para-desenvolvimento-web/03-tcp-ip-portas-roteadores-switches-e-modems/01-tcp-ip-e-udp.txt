TCP/IP e UDP

TCP/IP - Protocolo de comunicação entre dispositivos em rede.

TCP - Transmission Control Protocol - Protocolo de controle de transmissão.

IP - Internet Protocol - Protocolo de Internet


Modelo de camadas:

1. Física (ex. Placa de rede)
2. Rede (ex. IP)
3. Transporte (ex. Protocolos TCP, UDP)
4. Aplicação (ex. FTP, SMTP, HTTP)


TCP x UDP

O protocolo UDP é muito mais rápido, mas ele não verifica o retorno do dado enviado, exemplo é o Skype com uma reunião em vídeo, ele envia o vídeo mas não se importa se foi recebido.
